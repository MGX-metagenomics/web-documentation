News

2023-09-21: Public release of MGX 2.0, which supports metagenome coassembly, taxonomic binning and de novo metatranscriptome assembly.

2020-12-23: The Center for Biotechnology has discontinued its FTP service; benchmark datasets used in the MGX manuscript have been moved to Zenodo and can be retrieved using the DOI 10.5281/zenodo.4382271.

2020-04-06: With the most recent database update, the default taxonomic classification workflow employed by MGX is now also able to recognize and classify SARS-CoV-2 sequences.
